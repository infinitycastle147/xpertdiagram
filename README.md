# XpertDiagram

A comprehensive diagramming solution with both backend API and frontend interface.

## Structure

- `/backend` - Python/FastAPI backend service
- `/frontend` - Next.js frontend application

## Backend

The backend is a Python-based API service built with FastAPI that provides diagramming functionality.

### Setup
```bash
cd backend
pip install -r requirements.txt
```

### Running
```bash
uvicorn main:app --reload
```

## Frontend

The frontend is a Next.js application that provides an interactive interface for creating and managing diagrams.

### Setup
```bash
cd frontend
npm install
```

### Running
```bash
npm run dev
```

## Features

- Diagram creation and editing
- Multiple diagram types supported
- Real-time collaboration (if implemented)
- Export/import capabilities
- Customizable shapes and connectors

## Development

Both backend and frontend can be developed independently. Make sure to check the respective directories for specific development instructions.
